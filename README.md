# Challenge-amigo-secreto

Este proyecto permite gestionar una lista de amigos y realizar un sorteo aleatorio para determinar quién será el "amigo secreto".

Cómo Usarlo:
1- Agrega nombres de amigos a la lista.
2- Verifica que haya al menos dos amigos registrados.
3- Haz clic en el botón de sorteo para seleccionar un "amigo secreto".
4- Se mostrará el resultado en pantalla.
